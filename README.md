# Eta
A simple and very much WIP open-source rogue-like dungeon crawler

## Features
* Randomly Generated Levels
  * Currently 3 different 'structures'
* Items
  * 2 Weapons
  * 3 Scrolls
  * 1 Potion
  * 1 Shield
* Monsters
  * 2 of em!
* Basic leveling and xp system
* Messaging System
* Spell radius display
* Probably something else
